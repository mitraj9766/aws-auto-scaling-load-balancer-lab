# AWS Auto Scaling and Application Load Balancer Lab

This repository contains a hands-on AWS lab demonstrating the deployment of a scalable Windows web application using Amazon EC2, Amazon Machine Image (AMI), Application Load Balancer (ALB), and Auto Scaling.

The project shows how a Windows IIS web server can automatically scale during increased traffic while maintaining high availability through load balancing.

---

## Lab Environment

- Amazon Web Services (AWS)
- Amazon EC2
- Windows Server
- Internet Information Services (IIS)
- Amazon Machine Image (AMI)
- Application Load Balancer (ALB)
- Auto Scaling Group
- Security Groups
- Amazon VPC

---

## Project Workflow

1. Launch a Windows EC2 instance.
2. Configure IIS and deploy a sample website.
3. Create a custom AMI.
4. Launch additional instances from the AMI.
5. Configure an Application Load Balancer.
6. Create and configure Target Groups.
7. Verify load balancing.
8. Create a Launch Template.
9. Configure an Auto Scaling Group.
10. Verify automatic instance creation and traffic distribution.
